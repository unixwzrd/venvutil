# Function: general_help
 `general_help` - Display general help options for the 'help' command.
## Description
- **Purpose**:
  - Provide an overview of the available help commands.
- **Usage**: 
  - `general_help`
- **Scope**:
  - Global
- **Input Parameters**: 
  - None
- **Output**: 
  - Lists the general help commands available.
- **Exceptions**: 
  - None
