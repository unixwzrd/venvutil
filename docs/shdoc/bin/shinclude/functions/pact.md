# Function: pact
`pact` - Switch to the Previous Active VENV.
## Description
- **Purpose**: 
  - Switches to the previous active virtual environment.
- **Usage**: 
  - `pact [-h]`
- **Options**: 
  - `-h`   Show this help message
- **Input Parameters**: 
  - None
- **Output**: 
  - Activates the previous VENV.
- **Exceptions**: 
  - Errors if no previous environment exists.
