# Function: snum
`snum` - Force set the VENV Sequence number.
## Description
- **Purpose**: 
  - Forces the setting of the VENV sequence number.
- **Usage**: 
  - `snum [number]`
- **Input Parameters**: 
  - `number` (integer) - The sequence number to set.
- **Output**: 
  - Updates the sequence number for the current VENV.
- **Exceptions**: 
  - None
