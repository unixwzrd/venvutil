`to_upper` - Convert a String to Uppercase
## Description
- **Purpose**:

- **Usage**: 
  - `upper_str=$(to_upper "hello")`
- **Input Parameters**: 
  - `str`: The string to convert.
- **Output**: 
  - The uppercase version of the input string.
- **Exceptions**: 
  - None.
