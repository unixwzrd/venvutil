# Function: vnum
`vnum` - Return the current VENV sequence number.
## Description
- **Purpose**: 
  - Returns the sequence number of the current virtual environment.
- **Usage**: 
  - `vnum`
- **Input Parameters**: 
  - None
- **Output**: 
  - The sequence number of the current VENV.
- **Exceptions**: 
  - None
